# 🩺 Medical Image Disease Detection System

## 📖 Project Overview

This project focuses on detecting diseases from chest X-ray images using Artificial Neural Networks (ANN). The model is trained to classify X-ray images into two categories: **Normal** and **Pneumonia**.

The project demonstrates the complete deep learning workflow, including image preprocessing, model training, evaluation, visualization, and prediction.

---

## 🚀 Features

- Load and preprocess chest X-ray images
- Resize images for model training
- Encode class labels
- Split dataset into training and testing sets
- Build an Artificial Neural Network (ANN)
- Train and validate the model
- Evaluate model performance
- Generate predictions
- Display Classification Report
- Display Confusion Matrix
- Plot Training Accuracy and Loss
- Save the trained model
