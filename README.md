# 🩺 Medical Image Disease Detection System

## 📖 Project Overview

This project focuses on detecting diseases from chest X-ray images using an Artificial Neural Network (ANN). The model is trained to classify chest X-ray images into two categories: **Normal** and **Pneumonia**.

The project demonstrates a complete deep learning workflow, including image preprocessing, model training, evaluation, visualization of performance metrics, and prediction on unseen chest X-ray images.

---

## 🚀 Features

- Image preprocessing and resizing
- Dataset preparation and label encoding
- Artificial Neural Network (ANN) model training
- Model evaluation using Accuracy Score
- Classification Report
- Confusion Matrix
- Training Accuracy & Loss Visualization
- Prediction on custom chest X-ray images
- Save the trained model for future use

## ▶️ How to Run

1. Clone this repository.

```bash
git clone <your-github-repository-link>
```

2. Install the required libraries.

```bash
pip install -r requirements.txt
```

3. Open the notebook in Google Colab or Jupyter Notebook.

4. Run all cells sequentially.

5. Upload a chest X-ray image to test the trained model.
